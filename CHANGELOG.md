# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-04-08

### Added
- HTTPS-only network calls for IP discovery and DuckDNS update.
- Retry + timeout strategy for network resilience.
- Fallback public IP providers.
- Local runtime logs and state persistence.
- Health status section in tray menu.
- "Test Config Now" action in tray menu.
- DPAPI token protection.
- Start-with-Windows option.
- Core validation rules + unit tests.

### Changed
- Settings UI modernized for better readability.
- Validation and IP filtering hardened.
- About dialog now shows dynamic assembly version.
